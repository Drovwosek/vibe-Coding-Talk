# Naming Checks

- Python modules use snake_case: openai_service.py, http_app.py, prompts.py.
- Python functions use verb phrases for actions: generate_post, research_venue, validate_venue_audience.
- Browser storage keys use the postovaya.<domain> prefix.
- Browser DOM references live under the els object and use short semantic names matching the UI role.
- Data attributes use kebab-case in HTML/CSS and camelCase through dataset in JS: data-remove-draft -> dataset.removeDraft.
- User-facing Russian messages should be concrete and actionable: say what to fill, connect, save, or retry.
